# Wendell Costa — Portfolio

Personal portfolio website built with React + TypeScript. Features a multilingual interface (EN, PT, ES, FR, DE), animated skill bars, expandable project rows, and a functional contact form.

## Tech Stack

- React + TypeScript
- Vite
- CSS custom properties (no UI framework)
- Formspree (contact form)

## Features

- Auto-detects visitor's browser language on load
- 5 languages: English, Portuguese, Spanish, French, German
- Scroll-triggered reveal animations
- Responsive layout
- Contact form with success modal

## Running locally
```bash
npm install
npm run dev
```
